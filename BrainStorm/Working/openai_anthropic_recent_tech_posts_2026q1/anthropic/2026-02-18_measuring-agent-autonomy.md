# Measuring AI agent autonomy in practice

- 机构：Anthropic
- 日期：2026-02-18
- 分类：Research
- 原文链接：https://www.anthropic.com/research/measuring-agent-autonomy
- 关键词：autonomy, measurement, claude code, deployment, human intervention

## 中文速览
- 这篇不是只谈 benchmark，而是尝试测量真实世界里 agent 的‘实际自治程度’。
- 文中用 Claude Code session 数据观察 turn duration、人工介入次数、以及经验用户如何更愿意 auto-approve，说明部署中的 autonomy 和实验室能力并不完全等价。
- 它对你做个人 agent / 本地助手很有参考价值，因为现实自治不仅受模型能力影响，也受产品设计、用户信任和工作流约束影响。
