## 中文速览
- 讲 agent eval 为什么比普通单轮模型评测难：因为 agent 有多轮、工具、环境状态变化，错误会传播和累积。
- 这篇比较系统地整理了 agent eval 的结构、grader 设计、以及为什么真实部署中的 eval 需要和系统复杂度一起升级。
- 如果你在做 Butler 的任务质量评估、回归检查和自动化验证，这篇非常值得收。
