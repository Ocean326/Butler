 Claude Code的自律系统  
 
 使用 everything-claude-code 中的 Hooks，就相当于给 Claude Code 装了一个「自律系统」。  
 
 每当 Claude 准备做某件事、或刚做完某件事，就会自动触发对应的脚本——格式化、类型检查、会话记忆……全部不需要我们开口。  
 
 everything-claude-code 是 Anthropic 黑客松冠军在实际开发中积累的配置，是 Claude Code 生态里最受欢迎的配置合集之一，在 Github 的 Star 目前已经超过 7.5 万。  
 
 我们按照一次写代码 session 的时间顺序，从头到尾过了一遍这些 hooks 的功能和调用时机。
