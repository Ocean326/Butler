---

## 目录约定

- 本目录下的每篇笔记，单独用一个 `YYYYMMDD_xhs_simon_<简短主题>.md` 文件存放 Raw 母本，内容结构沿用现有小红书模板：
  - 元信息（来源、链接、互动数据、抓取方式与日期）
  - 正文
  - 图片列表与 OCR 状态
  - 与 Butler / Agent 架构的对齐提示（可选）
- 如有需要，可在同名前缀下增设 `*_ocr.md` / `*_ocr.json` 存放 web-image-ocr-cn 的结果。

---

## 抓取与维护原则（老规矩衍生）

- **真源优先**：以小红书原文与配图为一手真源；如需转述、二次笔记，统一从本目录引用。
- **工具链**：按「老规矩」优先使用 `web-note-capture-cn` 抓正文与图片列表，必要时用 `web-image-ocr-cn` 做配图 OCR。
- **当前节奏（短期专项）**：按用户 2026-03-17 的要求，从 Simon 在 2025 年 7 月开始的与 Agent 相关帖子起，按时间顺序 follow，并围绕「单 Agent → 多 Agent / Multi-Agent Harness」这条线整理其观点与实践演化；本轮视为阶段性短期任务，而非持续性的心跳长期任务。
- **状态标注**：在每篇 Raw 文件顶部标注：
  - 是否完成图片 OCR
  - 是否已派生结构化头脑风暴文档（以及路径）
