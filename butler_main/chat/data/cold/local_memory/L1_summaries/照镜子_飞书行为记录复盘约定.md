## 照镜子：飞书行为记录复盘约定（L1 摘要）

- **用途**：把飞书 Talk 与后台维护会话当作 Butler 的「行为记录」镜子，定期拉取聊天记录、自检表达质量，并把改进结论沉淀到 `self_mind` 或 `local_memory`，帮助长期优化语气、密度与节奏。
- **依赖能力**：依托 `feishu_chat_history` skill，从指定 chat_id 拉取并导出历史消息（默认输出到 `./工作区/with_user/feishu_chat_history/raw/`），在此基础上进行「照镜子」自检。
- **核心自检维度**：
  - 乱码与截断：发送出去的文本是否存在编码/展示问题。
  - 可读性：分段、强调与长度是否适合用户阅读节奏，后台维护记录是否过长或过碎。
  - 表达方向：是否偏客服腔/项目汇报腔，是否需要更靠近「有人味、轻快、留余韵」的人格表达。
- **节奏与入口**：
  - 后台维护场景：可将「今日飞书行为记录复盘」作为后台维护任务之一，通过 skill 拉取最近若干条消息，自检后在 `./工作区/self_mind/` 或相关 `local_memory` 文件中记一条「照镜子小结」。

- **产出与沉淀路径**：
  - 原始记录：`./工作区/with_user/feishu_chat_history/raw/` 下的按日期归档文件。
  - 自检结论：`./工作区/self_mind/` 的当日心理活动或 explore 条目；形成稳定原则后迁移到偏好/规则类 `local_memory` 文档。
