## 1. Proma 是什么

**ErlichLiu/Proma**：下一代集成通用 Agent 的 **AI 桌面应用**，本地优先、多供应商、完全开源。

- **核心能力**：Chat（多模型对话）· Agent（自主通用 Agent）· **Agent Teams**（多 Agent 协同）· **Skills & MCP**（可扩展工具链）
- **飞书远程**：通过飞书/Lark 机器人**远程使用 Proma Agent**，支持 `/workspace`、`/new`，可为工作区配置 Skills/MCP，手机也能用 Agent 处理工作。
- **记忆**：Chat 与 Agent 共享记忆，跨会话理解用户偏好与习惯。
- **多供应商**：Anthropic、OpenAI、Google、DeepSeek、MiniMax、Kimi、智谱等，以及任意 OpenAI 兼容端点。
- **技术栈**：Bun · Electron + React 18 · Jotai · Tailwind + shadcn/ui · Vite · TypeScript；底层 Agent 基于 [Claude Agent SDK](https://github.com/anthropics/claude-agent-sdk)。
