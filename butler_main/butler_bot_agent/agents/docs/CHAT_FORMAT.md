# Chat 记录格式

统一格式：

`[HH:MM] [agent_name] 事项描述 | 输入: ... | 输出: ... | 下一步: ...`

最小合规要求：

1. 每个任务必须有 `开始` 记录。
2. 每个任务必须有 `结束` 记录（含结果与下一步）。
3. 若任务被阻塞，追加 `阻塞` 记录（原因 + 解除条件）。

示例：

`[09:30] [orchestrator_agent] 分发今日任务 | 输入: 昨日未完成2项 | 输出: 任务板V1 | 下一步: secretary更新日报`

`[14:20] [literature_agent] 更新文献卡3篇 | 输入: 关键词"轨迹重构 表征学习" | 输出: 文献卡_2026-03-05.md | 下一步: research_ops提炼研究空白`

`[16:40] [engineering_tracker_agent] 任务阻塞 | 输入: 训练日志中断 | 输出: 暂停里程碑更新 | 下一步: 等待环境修复后重跑`
