# 远景草稿

本目录用于存放 **Butler 的中长期方向草稿、阶段演进判断、概念框架、远景规划**。

## 定位

这里的文档和 `docs/daily-upgrade/` 不同：

- `daily-upgrade/`
  - 记录按日期归档的现状、阶段性改动、排查、实施计划
- `远景草稿/`
  - 记录 1.0 之后仍有效的中长期方向判断、概念型材料、产品/架构远景草稿

## 使用边界

1. 本目录优先放 **概念、框架、阶段判断、远景路线**。
2. 不在这里放当天的具体排障记录、一次性的实现细节、临时 diff 说明。
3. 若文档已经进入明确实施阶段，应回流到：
   - `docs/daily-upgrade/<MMDD>/` 记录阶段性计划与落地
   - `docs/concepts/` 沉淀长期稳定的正式概念文档
