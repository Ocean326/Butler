## 一句话裁决

从 `2026-04-05` 起，Butler Flow Desktop 的现役产品入口更新为：

- 默认先面对 `Manager 管理台`
- `New Flow` 只是空白 Manager thread，不再是独立旧分栏
- `History` 以 `Project Thread` 理解 manager session 与 supervisor flow，并同时保留两类 thread
- 创建前先在 Manager 内完成 brainstorm / requirements / delivery / test alignment
- 确认后通过 `manager-message -> manage_flow` 自动进入 `Supervisor` 流
- 在 Supervisor 流内点击 team agent，进入同一 thread 下的 `Agent focus`
- `Templates` 负责 `template + agent team` 的单流管理
- `Manager / New Flow` composer 默认小、底部 dock、随输入增长且上限不超过视口 1/3
- bridge 缺失时 renderer 显式进入 `Desktop bridge 未连接` 产品态，不再打 query/toast 风暴
- renderer 全面切换为冷白/蓝灰日间主题与蓝黑夜间主题，夜间为默认主题
