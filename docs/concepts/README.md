# Concepts

`docs/concepts/` 现在只保留三类材料：

1. 长期原则
2. 仍有效的概念模型
3. 当前仍可用的接入说明

历史设计稿、旧术语快照、退役方案统一迁到 [`history/`](./history/README.md)。

## 当前使用规则

1. 改动前不要直接整包扫 `concepts/`
2. 先读 [`docs/project-map/`](../project-map/README.md)
3. 只有在 `project-map` 或 `runtime` 明确指向时，再回到这里补长期背景

## 新文档约定

新建或补修的概念文档，头部统一包含以下字段：

- `状态`
- `类型`
- `最后核对日期`
- `当前替代入口`
- `是否允许作为改动依据`
