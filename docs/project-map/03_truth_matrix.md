1. 先判事实类型，再读文档
2. 一类事实只认一个主真源
3. 若主真源与代码、最近通过测试冲突，先以代码和测试为准，再回写文档
4. 任何涉及运行面、入口面或控制面的升级，在代码和测试之后还必须补一次实际重启复验
